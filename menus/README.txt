In this folder, we keep all python scripts that are responible for displaying menus.
Each script has a main() function that needs to be called to display the menu.

__init__.py - Import all menu modules for easy importing from other files
about.py - Handles about Menu.
howto.py - Handles Chess howto Menu.
loadgame.py - Handles loadgame menu, also defines utility functions for scanning, loading and deleting games.
online.py - Handles online Menu.
pref.py - Handles preferences menu, also defines utility functions for loading and saving user preferences.
singleplayer.py - Handles singleplayer menu
stockfish.py - Handles stockfish menu, also defines functions to handle stockfish install guides.
timer.py - Handles timer menu.
